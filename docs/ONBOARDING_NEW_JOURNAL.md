# Onboarding New Journals on Research Africa

This guide explains the process for onboarding new academic journals onto the Research Africa platform. Follow these steps to ensure a smooth setup and integration.

## 1. Prerequisites

-   Admin access to the Research Africa platform
-   Journal details (name, acronym, ISSN, website, scope, editorial board, etc.)
-   Logo and branding assets (optional but recommended)

## 2. Create a Journal Category

1. Log in as an admin.
2. Navigate to **Journals > Add New Journal** or **Categories > Add Journal**.
3. Fill in the required fields:
    - Journal Name
    - Acronym
    - ISSN (print/online)
    - Journal URL
    - Scope/Description
    - Select **Is Journal** (checkbox or toggle)
    - Upload logo (optional)
    - Set status to **Active**
4. Save the journal category.

## 3. Configure Journal Settings

-   Set up journal-specific settings such as:
    -   Submission guidelines
    -   Peer review workflow
    -   Email templates
    -   Theme and branding (if supported)

## 4. Add Editorial Board Members

1. Go to the new journal's page or **Editorial Board** section.
2. Add members by searching for existing users or creating new member profiles.
3. Assign roles (Editor-in-Chief, Associate Editor, Reviewer, etc.) and set their terms.

## 5. Assign Journal Roles and Permissions

-   Ensure editors and reviewers are assigned the correct roles and permissions for the journal.
-   Use the **Roles/Permissions** module to manage access.

## 6. Test Journal Access

-   Log in as an editor or reviewer to verify:
    -   Access to the correct journal dashboard
    -   Ability to manage submissions, assign reviewers, and communicate with authors

## 7. Launch and Announce

-   Once setup is complete, announce the new journal to the community.
-   Provide training or documentation to the editorial team as needed.

## 8. Ongoing Support

-   Monitor submissions and editorial activity.
-   Provide technical support and updates as required.
