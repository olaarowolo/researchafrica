# Assigning Peer Reviewers on Research Africa

This guide explains how to assign peer reviewers to manuscript submissions within the Research Africa platform.

## 1. Prerequisites

-   Editor or admin access to the journal
-   Manuscript(s) submitted to the journal
-   Reviewer profiles created in the system

## 2. Access the Manuscript Submission

1. Log in as an editor or admin.
2. Navigate to the **Journal Dashboard**.
3. Go to **Submissions** or **Manuscripts**.
4. Select the manuscript you wish to assign reviewers to.

## 3. Select Reviewers

1. In the manuscript details page, locate the **Peer Review** or **Assign Reviewers** section.
2. Click **Assign Reviewer** or similar action button.
3. Search for reviewers by name, expertise, or keywords.
4. Select one or more reviewers from the list.
5. Optionally, set a due date or add instructions for the reviewer.

## 4. Send Review Invitations

-   Click **Send Invitation** or **Assign** to notify the selected reviewers.
-   The system will send an email or in-platform notification to each assigned reviewer.

## 5. Track Review Progress

-   Monitor the status of each review (invited, accepted, in progress, completed) from the manuscript's review panel.
-   Send reminders if needed.

## 6. Manage Reviewer Responses

-   Reviewers can accept or decline the invitation.
-   If declined, assign a new reviewer as needed.

## 7. Complete the Review Cycle

-   Once all reviews are submitted, proceed with editorial decisions (accept, revise, reject, etc.).
